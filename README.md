# CS5 Adafruit RP2040 QT Trinkey (CircuitPython)

CircuitPython files to drive various controllers and other peripherals over STEMMA QT with the Adafruit QT2040 Trinkey.

## Tested Setup
- Adafruit QT2040 Trinkey
- Adafruit Wii Nunchuck Breakout Adapter
- Adafruit Wii Nunchuck
- Adafruit Mini Gamepad

## Installation/Usage
1. Run `transfer.py` to copy the repository onto the CIRCUITPY drive and open the miniterm.
